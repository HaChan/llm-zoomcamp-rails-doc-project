Sucker Punch adapter for Active JobSucker Punch is a single-process Ruby asynchronous processing library. This reduces the cost of hosting on a service like Heroku along with the memory footprint of having to maintain additional jobs if hosting on a dedicated server. All queues can run within a single application (e.g. Rails, Sinatra, etc.) process.Read more about Sucker Punchhere.To use Sucker Punch set the queue_adapter config to:sucker_punch.Rails.application.config.active_job.queue_adapter = :sucker_punch
