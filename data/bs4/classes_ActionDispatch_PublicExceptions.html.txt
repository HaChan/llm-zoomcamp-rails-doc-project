Action DispatchPublicExceptionsWhen called, this middleware renders an error page. By default if an HTML response is expected it will render static error pages from the/publicdirectory. For example when this middleware receives a 500 response it will render the template found in/public/500.html. If an internationalized locale is set, this middleware will attempt to render the template in/public/500.<locale>.html.
